# Molecules Components

Esta carpeta contiene componentes moleculares según el patrón Atomic Design.

Las moléculas son combinaciones de átomos que forman componentes más complejos pero aún reutilizables.

## Ejemplos:
- SearchBox (Input + Button)
- FormField (Label + Input + ErrorMessage)
- Card (Image + Title + Description)
- NavigationItem (Icon + Link)
